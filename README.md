Island Ecosystem Simulation

Проект представляет собой симуляцию экосистемы острова. На острове есть хищники, травоядные и растения. Животные могут есть, размножаться, перемещаться и умирать от голода.

##  Описание проекта

Остров представлен в виде локации. Каждая локация содержит:

- **Растения** — растут со временем, служат пищей для травоядных
- **Животные**:
  - **Хищники**: Волк 🐺, Удав 🐍, Лиса 🦊, Орёл 🦅
  - **Травоядные**: Утка 🦆, Лошадь 🐎, Корова 🐃, Кролик 🐇, Овца 🐑, Хомяк 🐁

### Основные возможности

- Животные могут **есть** (хищники — травоядных, травоядные — растения)
- Животные могут **размножаться** (при достаточном уровне сытости)
- Животные **умирают** от голода или будучи съеденными

##  Запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/Nokreen/islandkt.git
cd islandkt
2. Открыть в IntelliJ IDEA
Откройте IDEA → Open → выберите папку islandkt

IDEA автоматически подхватит структуру проекта

3. Добавить JUnit в classpath (если необходимо)
JAR-файлы уже лежат в папке lib. Если тесты не запускаются:

File → Project Structure → Modules → Dependencies

Нажать + → JARs or directories

Выберите все JAR-файлы из папки lib

4. Запустить приложение
Откройте Main.java

Нажмите зелёную стрелку 

Появится окно с симуляцией. Нажмите Start — остров начнёт "жить".
Запуск тестов
Запуск всех тестов
Нажмите правой кнопкой на папку src/test/java

Выберите Run 'All Tests'

Запуск отдельного теста
Откройте любой тестовый класс (например, AnimalTest.java)

Нажмите зелёную стрелку рядом с именем класса
