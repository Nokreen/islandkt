Island Ecosystem Simulation

Проект представляет собой симуляцию экосистемы острова. На острове есть хищники, травоядные и растения. Животные могут есть, размножаться, перемещаться и умирать от голода.

## 📖 Описание проекта

Остров представлен в виде сетки локаций (20×10). Каждая локация содержит:

- **Растения** — растут со временем, служат пищей для травоядных
- **Животные**:
  - **Хищники**: Волк 🐺, Удав 🐍, Лиса 🦊, Орёл 🦅
  - **Травоядные**: Утка 🦆, Лошадь 🐎, Корова 🐃, Кролик 🐇, Овца 🐑, Хомяк 🐁

### Основные возможности

- Животные могут **есть** (хищники — травоядных, травоядные — растения)
- Животные могут **размножаться** (при достаточном уровне сытости)
- Животные могут **перемещаться** по соседним клеткам
- Животные **умирают** от голода или будучи съеденными
